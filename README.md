# Assingment 22-10-2023

## Installation

Install with npm

```bash
  npm i
```

## Deploy

To run this project run

```bash
  npm run start
```

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`DB_URL`
`PORT`
`SECRET_KEY`
